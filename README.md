# Recursive Discovery Agent

This project explores autonomous scientific discovery through recursive AI agents. It implements a system where AI agents can recursively design, execute, and analyze experiments, aiming to accelerate hypothesis generation and validation in research.

## Key Features
*   Autonomous agentic workflows for designing and running computational experiments.
*   Recursive task decomposition, allowing agents to break down complex problems.
*   Integrated tools for data analysis, literature review, and hypothesis evaluation.
*   Modular architecture for customizing agent capabilities and knowledge domains.

## Tech Stack
*   Python
*   LangChain / LangGraph
*   OpenAI API (or other LLM providers)
*   Docker (optional)

## Getting Started
1.  Clone the repository: `git clone https://github.com/zoreanuj/recursive-discovery-agent.git`
2.  Install dependencies: `pip install -r requirements.txt`
3.  Set your API key: `export OPENAI_API_KEY='your-key'`
4.  Run the basic agent: `python main.py`
